Lesson 1:The Basics Of Web And HTML

How The Web Works

The Web is a bunch of computers that communicate with each other. When we(our browsers) makes request to the server through internet with the help of http protocol,server finds the appropriate HTML document and sends it back to the user's computer where a web browser interprets the page and displays it on the user's screen. This link will take you to the video which will explain you more in detail.

HTML Basics

HTML is a standard markup to create web pages and controls the structure of Web page.HTML stans for Hypertext Markup Language.HTML consists of text content,markup,references to other document and link to other pages.This link will take you to the video which will explain you more in detail.

Tags And Elements

HTML documents are made of HTML elements. When writing HTML, we tell browsers the type of each element by using HTML tags.For example,b tag is used for making the text bold.This link will take you to the video which will explain you more in detail.

Computers Are Stupid

Computers are stupid because they interpret instructions literally. This makes them very unforgiving since a small mistake by a programmer can cause huge problems in a program.

Inline vs Block

HTML elements are either inline or block. Block elements form an invisible box around the content inside of them.Some of the block elements are p,form and div elements.Examples of inline elements are a,br,img,b and many more.This link will take you to the video which will explain you more in detail.

Lesson 2:Creating A Structured Document

The "tree-like structure" of HTML

"Tree-like structure" comes from the fact that HTML elements can have other elements inside them.We can make HTML document structured by using many tags,one of the example being div tag.Browses turn the HTML tag into elements which forms a tree-like structure called DOM tree.

HTML-CSS-DOM

HTML:It controls the structure of web page.
CSS:It controls the style of the web page.
DOM(Document Object Model):Browsers turn the HTML tags into elements that forms a tree like structure called DOM.This video will explain you more in detail.

Lesson 3:Box Sizing and Positioning

Box Sizing

HTML elements are boxes.Because there are so many component to each box,sometimes it becomes difficult to get the size of box right.So there are two techniques we can use to deal with it:

  • Set sizes in terms of percentages rather than pixcels.
  • Set the box-sizing attribute to border-box for every element.
Box positioning

As divs are block elements,so by default they take up the entire width of a page.Adding the rule,display:flex to the CSS file will override the behavior and let the divs appear next to each other.
